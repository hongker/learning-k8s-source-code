

# learning-k8s-source-code

从源码角度出发，学习k8s的原理。

目前打算以`kube-apiserver` `kube-controller-manager`  `kube-scheduler`  `kubelet` `proxy` 和 `kubectl`  这6个组件为主线进行源码级别的学习。

同时还顺便记录一些平时用到和k8s相关的知识，例如etcd, docker, linux等相关知识。
